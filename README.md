# alarm_clock-V2.0
version 2.0
1. 新增暂停 / 恢复按钮
添加了 paused 布尔变量跟踪暂停状态
在按钮框架中新增了暂停 / 恢复按钮，点击可切换状态
按钮文字和颜色会根据状态变化（暂停：红色，恢复：青色）
check_alarm 方法中增加了暂停状态判断
恢复时会重置计时，避免暂停期间累积的时间触发立即提醒
2. 弹窗置顶功能
重写了弹窗显示逻辑，使用 Toplevel 创建自定义弹窗
设置 attributes("-topmost", True) 确保弹窗置顶
添加 focus_force() 和 grab_set() 确保弹窗获得焦点且为模态窗口
保留了 messagebox 作为备用方案，确保兼容性
3. 弹窗和声音同步
修改 trigger_alarm 方法，使用多线程同步执行弹窗和声音
创建 _execute_alert 方法管理所有提醒操作的线程
确保弹窗和声音同时开始播放，而非串行执行
使用 root.after(0) 确保 UI 更新在主线程执行，避免线程安全问题
4. 其他优化
调整窗口高度以容纳新按钮
优化状态提示文本
确保所有 UI 操作都在主线程执行
增强了异常处理，提高程序稳定性
使用说明：
暂停 / 恢复按钮：点击可暂停提醒，再次点击恢复（恢复后重新开始计时）
弹窗始终置顶：无论有多少窗口打开，提醒弹窗都会显示在最上层
声音和弹窗同步：触发提醒时，声音和弹窗会同时出现
